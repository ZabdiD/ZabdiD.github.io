Módulos Adicionales

Centralizador de Datos

  1. Posibilidad de generar y transmitir por modem los movimientos de cada sucursal y centralizar en una oficina.

  2. Permite realizar un seguimiento y control de cada una de las sucursales desde un solo lugar.

  3. Control de ventas diarias y mensuales.

  4. Liquidación de Obras Sociales con los datos particulares de cada sucursal.

  5. Evaluación y estadística.

Módulo de Control de Stock

  1. Permite generar reportes de control diario, semanal y mensual de movimiento de stock de productos.

  2. Reporte y seguimiento de productos determinados.

  3. Control de Facturación.

Gestión de Productos, Obras Sociales y Cuentas Corrientes

  1. Este módulo permite importar registros de los archivos maestros de Productos, Obras Sociales y Cuentas Corrientes y luego exportar los datos a cada una de las sucursales.

Módulo para Validación de Recetas ON-Line

  1. Estos módulos permiten la operación de validación de recetas On Line previa a la facturación, otorgando al operador estado de validación de datos y código de autorización.

Módulo para Gestión de Caja

  1. Este programa se instala en una PC junto al cajero para posibilitar el control de los tickets emitidos y cobrados.

  2. Reporte de tickets emitidos y que no fueron cobrados.

  3. Estado de la caja actual y anteriores.

  4. Estado de posibles diferencias de caja, entre lo cobrado y lo facturado.
